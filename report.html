How we believe we have met the expectations:

  1. Client: We have a real client being the African Student Asscociation organization on campus
  2. Header: Each webpage has the same header throughout using a global stylesheet
  3. Pages: There was 10 original pages that was planned out. I believe 9/10 of those pages were created, 1 missing due to contribution issues.
  4. Nav: Included navigation links as part of the headers on each webpage.
  5. Responsive: Used media queries to implement seperate layouts for large screens and small screens. Our nav bar turns into a hamburger menu when small enough.
  6. Global Style Sheet: Our global style makes sure the header and footer is the same throughout all pages. We have a neutral color scheme going on through out the entire website, and each page contains their own stylesheet.
  7. Alt Attribute: Every image has an alt attribute that describes the image content
  8. Favicon:
  9. Non-HTML Resources: Every image in each page is held within the images folder. Created a CSS folder to store all CSS files.
  10. Third-Party Libraries or Frameworks: No libraries and frameworks are used within the website.
  11. W3C Validator: Each page has been validated by W3C, if there were errors before there are no more now.
  12. Sharing Project: The assignment will be shared through a .zip file on Canvas.
  13. Naming Convention: All files are lowercase and contain only letters.
  14. Alignment: All content is neatly organized within each file.
  15. Font Faces: Used two different font faces across the webpage. Headings and footer have a different front from the other content on the page
  16. External Links: There are links to externals sites like social media all throughout the site. In the footer there is links to Facebook and Instagram. On the E-board page, each member as links to their own social media like Instagram, Snapchat, and Twitter. Clicking on these links opens a new tab instead of replacing the current page.
  17. Source Code: All source code is legible and easy to read. Used meaningful names for classes and IDs
  18. Nav Bar: The navbar is consistent throughout the entire website site because of the global.css stylings shared in each file.
  19. Header and Footer: Each webpage has the same header and footer.
  20. Flexboxes/Grids: There's grids and flexboxes implemented in almost every, if not every, page on the site.

Critique from client

I appreciate the way the photo and quotes on the home page captures the essence of ASA. The fun facts page adds an engaging element to the website. However, it would be great if the fun facts page could be more interactive, maybe adding a "guessing" game or other interactive features. It would also be helpful if the events page had a calender view to provide a clear overview of upcoming events.

In terms of design, it would be nice to incoporate more vibrant colors or African-inspired designs to tie in with the theme. I like that the website provides enough information about ASA and its various aspects. It would be great if the website could be regularly updated to ensure it remains relevant and current for future years.

Overall I appreciate all of the time and effort put into creating this website but there are opportunities for improvement in terms of interactiviy, design and updates.
